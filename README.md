# ListingGen Pro

A lightweight AI-powered tool for generating e-commerce product listings from CSV files.

## Features
- Batch generate product listings using LLM
- CSV input / TXT & CSV output
- Modular project structure (parser / AI / exporter / UI)

## Tech Stack
- Python
- LLM API (Volcengine Ark)
- customtkinter

## Status
- Actively under development
